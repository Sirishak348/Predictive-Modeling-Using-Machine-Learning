# Predictive Modeling Using Machine Learning

## Project Overview
This project predicts passenger survival using the Titanic dataset by implementing a Decision Tree Classifier.

## Dataset
- Titanic Dataset

## Algorithm Used
- Decision Tree Classifier

## Project Workflow
- Data Loading and Exploration
- Handling Missing Values
- Feature Selection
- Encoding Categorical Variables
- Train-Test Split
- Model Training
- Prediction and Evaluation

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- ROC Curve (AUC Score)
- Decision Tree Visualization

## Conclusion
This project provided practical experience in predictive modeling and machine learning model evaluation using Python and Scikit-learn.
